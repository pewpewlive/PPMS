# PPMS
The most advanced mesh editor for PewPew Live out there.

Powered by React, Vite, TypeScript and React Three Fiber.

**⚠️ The project is currently under heavy development - expect bugs and missing features!**

Found a bug? Have a suggestion? Make an issue!

## Setting up a local dev environment
Follow these steps:

```sh
git clone https://github.com/pewpewlive/PPMS.git
cd PPMS
npm install
npm run dev
```
